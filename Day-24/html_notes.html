HTML Structure 1. What is HTML? HTML (HyperText Markup Language) is the standard markup language used to create web pages. HTML is used to: - Structure web pages - Display text, images, videos, links, tables, forms, etc. - Define the content of a webpage HTML is not a programming language; it is a markup language. -------------------------------------------------- 2. Difference Between Head and Body <head> - Contains metadata - Not visible on webpage - Contains title, CSS, scripts, etc. <body> - Contains visible webpage content - Displayed in browser -------------------------------------------------- 20. HTML Best Practices - Always include <!DOCTYPE html>. - Use proper indentation. - Use meaningful tag names. - Close all non-void tags. - Keep code organized and readable. - Use comments where necessary. -------------------------------------------------- 21. Common Interview Questions Q1. What is HTML? HTML is a markup language used to structure web pages. -------------------------------- Q2. What is the purpose of <!DOCTYPE html>? It tells the browser that the document uses HTML5. -------------------------------- Q3. What is the difference between <head> and <body>? <head> contains metadata. <body> contains visible webpage content. -------------------------------- Q4. What are HTML elements? Elements consist of an opening tag, content, and a closing tag. -------------------------------- Q5. What are HTML attributes? Attributes provide additional information about HTML elements. -------------------------------- Q6. What are empty elements? Elements that do not require a closing tag. Examples: <br> <hr> <img> <input> -------------------------------------------------- 22.
